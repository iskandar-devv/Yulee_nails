# Yulee_nails - Студия маникюра

Профессиональный одностраничный сайт для студии маникюра и педикюра.

## 🎨 Описание проекта

Современный, полностью адаптивный сайт-визитка для nail-студии с элегантным дизайном в пастельных тонах. Включает все необходимые секции для представления услуг, портфолио работ, информации о мастерах и онлайн-записи.

## 🚀 Особенности

- ✨ Современный минималистичный дизайн
- 📱 Полная адаптивность (mobile-first)
- 🎭 Плавные анимации и переходы
- 🖼️ Интерактивная галерея с фильтрацией
- 💅 Lightbox для просмотра работ
- 📝 Форма онлайн-записи с валидацией
- 🎠 Слайдеры отзывов и мастеров
- 🔝 Кнопка быстрого возврата наверх
- 📊 Анимированная статистика
- 🎯 SEO-оптимизация
- ♿ Семантическая разметка
- ⚡ Быстрая загрузка
- 🌐 Кроссбраузерность

## 🛠️ Технологии

- **HTML5** - семантическая разметка
- **CSS3** - современные стили, Flexbox, Grid, анимации
- **JavaScript (ES6+)** - интерактивность без фреймворков
- **Font Awesome** - иконки
- **Google Fonts** - типографика (Playfair Display, Montserrat)

## 🎯 Секции сайта

1. **Header** - навигация с фиксированной шапкой
2. **Hero** - главный экран с призывом к действию
3. **О нас** - информация о студии, преимущества, статистика
4. **Услуги** - карточки с описанием услуг и ценами
5. **Галерея** - портфолио работ с фильтрацией
6. **Мастера** - команда специалистов
7. **Прайс-лист** - детальная таблица цен
8. **Отзывы** - слайдер с отзывами клиентов
9. **Запись** - форма онлайн-бронирования
10. **Контакты** - информация и карта
11. **Footer** - дополнительная информация

## 💻 Как запустить

### Локально

1. Скачайте все файлы проекта
2. Откройте `index.html` в любом современном браузере
3. Готово! Сайт работает без необходимости установки зависимостей

### На GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы проекта
3. Перейдите в Settings → Pages
4. Выберите ветку `main` и папку `root`
5. Сохраните изменения
6. Сайт будет доступен по адресу: `https://username.github.io/repository-name`

### На хостинге

1. Загрузите все файлы на хостинг через FTP/SFTP
2. Убедитесь, что `index.html` находится в корневой директории
3. Сайт готов к работе
